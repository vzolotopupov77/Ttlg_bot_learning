# Итерация 8 — Тестирование: план

## Цель

Актуализировать ручные тест-сценарии для 5 экранов; внедрить Vitest, Testing Library и MSW; покрыть ключевые компоненты и формы автотестами; цель `make frontend-test` проходит.

## Ценность

Регрессии на UI ловятся автоматически; ручная приёмка опирается на чек-листы; backend в тестах не обязателен (моки/MSW).

## Предусловия

| Предусловие | Статус |
|-------------|--------|
| Экраны 1–5 реализованы (итерации 2–6, ревью 7) | ✅ |
| Контракты API зафиксированы | `docs/tech/api-contracts.md` |
