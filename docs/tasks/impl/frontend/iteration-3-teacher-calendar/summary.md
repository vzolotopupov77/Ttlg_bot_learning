## Что сделано

- Экран `/teacher/calendar`: недельная сетка (7 дней), навигация по неделям (`week_start` в URL), карточки занятий с легендой флагов.
- `LessonDialog`: создание/редактирование/удаление занятий (`POST`/`PUT`/`DELETE /v1/lessons`), флаги через `PATCH .../flags` с откатом при ошибке; заглушки кнопок «Отправить ДЗ» / «Прикрепить файл» / «Напомнить».

- Общие модули: `lib/api-server.ts`, `lib/api-client.ts`, `lib/calendar-dates.ts`, `lib/types/teacher-calendar.ts`; `<Toaster />` в `providers.tsx`.
