# Итерация 7 — Ревью качества frontend: план

## Цель

Провести ревью кода `frontend/` по best practices Next.js App Router и React; устранить критические замечания; **внедрить цветовую палитру из макетов** в семантические токены shadcn (`globals.css`), чтобы все экраны визуально соответствовали задуманному UI до финального тестирования (итерация 8).

## Ценность

Единый технический долг по качеству закрыт; тема не остаётся нейтральной по умолчанию — акценты и фоны согласованы с макетами форм; меньше правок «на глаз» на этапе приёмки.
