# Задача 12: Ревью и доработка репозиториев — план

**Дата:** 2026-04-05

## Цель

Проверить 5 репозиториев на паттерны `None`-handling, корректное использование `scalars()`, отсутствие утечки сессии, DI-совместимость. Применить паттерны skill `fastapi-templates`.

## Чек-лист по репозиторию

- `get_by_id` возвращает `None` (не кидает исключение)
- `scalars().all()` для списков, `scalar_one_or_none()` для одного
- `session` принимается первым аргументом, не создаётся внутри
- `flush()` в репозиториях, `commit()` — в сервисе/роутере
