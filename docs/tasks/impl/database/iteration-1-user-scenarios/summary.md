## Ключевые выводы для итерации 2

1. **Сущности `Topic` и `Material` отсутствуют в схеме** — упомянуты в `vision.md`, нужны для сценария SC-T-05 и LLM-контекста в SC-S-03. Необходимо добавить при проектировании схемы.

2. **Все 6 текущих сущностей** (`User`, `Lesson`, `Assignment`, `Progress`, `Dialogue`, `Message`) покрывают сценарии ученика и базовые сценарии преподавателя.

3. **`Progress` — агрегат**: необходимо уточнить механизм пересчёта (вручную или по событию) и зафиксировать в ADR.

4. **Канал `web` в `Dialogue`** используется в сценарии SC-S-03 — при реализации веб-клиента нужно передавать `channel=web`.
